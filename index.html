SPESIFIKASI SOAL

1. TAMPILAN ANTARMUKA (UI)
  • Buatlah 2 Input Field menggunakan widget TextField.
  • Set properti keyboardType menjadi TextInputType.number.
  • Tambahkan decoration (hintText) agar jelas (misal: "Angka 1").
  • Buat 1 Tombol menggunakan ElevatedButton dengan teks "HITUNG".
2. LOGIKA & FUNGSI
  • Wajib menggunakan TextEditingController untuk mengambil input.
  • Gunakan double.tryParse() untuk mengubah teks menjadi angka dengan aman.
  • Lakukan operasi Penjumlahan (+).
  • Wajib menggunakan setState() untuk memperbarui hasil di layar.
